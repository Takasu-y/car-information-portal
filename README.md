# car-information-portal

自動車の情報を検索することができるサイトです。

# Live Demo
https://car-information-portal.herokuapp.com/

# Usage
* サイト上部のformからメーカー、モデル、年式を入力すると該当する車の情報が表示されます。  
* トップページのメーカーロゴをクリックすると、対象メーカーのモデル一覧が表示されます。
